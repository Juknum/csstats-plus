CSStats+ enhances CSStats with a cleaner interface and clearer match data.

- Full map names with better icons
- Sharper rank visuals and clearer rank progression

- SVG icons used where available (maps and ranks)

- More complete map and profile stats display
- Better visual consistency across profile and match pages
